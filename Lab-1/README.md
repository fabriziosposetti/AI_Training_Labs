# URL Shortener

A fast, simple, and beautiful URL shortener built with FastAPI and modern web technologies.

## Features

- ⚡ **Fast & Efficient** - Built with FastAPI for lightning-fast performance
- 🎨 **Beautiful UI** - Modern, responsive design with smooth animations
- 🔒 **Secure** - Input validation and safe URL handling
- 📊 **Smart Storage** - SQLite database with duplicate detection
- 🎯 **Simple API** - Easy-to-use REST API endpoints
- 📱 **Mobile Friendly** - Fully responsive design
